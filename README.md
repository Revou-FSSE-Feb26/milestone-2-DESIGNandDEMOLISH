# RevoFun 

RevoFun is a mini game website built using HTML, Tailwind CSS, and JavaScript.
This project contains multiple interactive browser games designed to deliver a fun and simple gaming experience with responsive layouts and leaderboard systems.

---

## Features

* Responsive modern UI using Tailwind CSS
* Interactive mini games
* Local leaderboard system using LocalStorage
* Animated homepage sections
* Simple and beginner-friendly project structure

---

## Games Included

### 1. Clicker Game

Test your reflex speed by clicking as fast as possible before the timer runs out.
Earn bonus time at specific score milestones and compete for the highest leaderboard score.

Features

* Countdown timer
* Bonus time rewards
* Score tracking
* Local leaderboard



### 2. Number Guessing Game

Guess the secret number between 1 and 100 within limited chances.
Use logic and hints to find the correct answer and maximize your score.

Features

* Random number generation
* Hint system
* Chance tracking
* Score calculation
* Local leaderboard



### 3. Rock Paper Scissor

Play the classic Rock Paper Scissor game against the computer.
Reach the target score before the computer to win the match.

Features

* Real-time score updates
* Computer random choices
* Match result display
* Local leaderboard
